# Conceptual Framework

## ปัจจัยที่ส่งผลต่อความเสี่ยงไฟป่าในจังหวัดเชียงใหม่

1. ความร้อน (Heat)

ในที่นี้ใช้เป็นความร้อน Land Surface Temperature เป็นตัวชี้วัดความพร้อมติดไฟของพื้นที่ ยิ่งความร้อนสูงยิ่งเสี่ยงติดไฟ
   
2. ความหนาแน่นเชื้อเพลิง (Fuel Density)

ใช้ NDVI ในการดูพื้นที่มีพืชปกคลุมหนาแน่น หรือ Vegetation Biomass สูง = มีเชื้อเพลิงเยอะ ใช้เพิ่มความเร็วในการลุกลามและขนาดไฟ
   
3. ความแห้งของเชื้อเพลิง (Dryness)

ใช้ NDMI ในการดูลักษณะของเชื้อเพลิง หากเชื้อเพลิงยิ่งแห้งมากจะติดไฟง่ายกว่าเชื้อเพลิงที่ชื้น
   
4. ประวัติไฟ (History)

เพื่อให้โมเดลสามารถดู pattern ของพื้นที่ที่ไฟเกิดซ้ำ เช่น พื้นที่แห้ง, ใกล้ชุมชน, มีพืชที่ติดไฟง่ายและเรียนรู้ว่าพื้นที่ไหนเสี่ยงเกิดไฟป่า

## น้ำหนักของแต่ละปัจจัย
มาจากการหา feature importance โดยจะให้ความสำคัญตัวละ 50% แล้วหาออกมาว่าตัวไหนมีค่า weight เยอะที่สุด โดยได้ค่าน้ำหนักแต่ละปัจจัยออกมาดังนี้ 
* Dryness = 0.458
* Heat = 0.539
* Fuel Density = 0.424
* History = 0.412

เมื่อทำให้ค่าเป็น 1 จะได้ดังนี้ weight = [0.25, 0.30, 0.23, 0.22]  เรียงลำดับตาม [Dryness, Heat, Fuel Density, History]

## Validate

ทำการ validate ด้วยข้อมูล MODIS Fire Burned Area
