## 1. ข้อมูลเบื้องต้นของการทำโมเดล

1.1 การแบ่งที่ดิน
แบ่งเป็น 5 คลาส ได้แก่
1. เกษตร
2. ป่า
3. เมือง
4. น้ำ
5. พื้นที่เบ็ดเตล็ด (อื่นๆ)

1.2 Training Samples

ใช้ข้อมูลจาก ESA World cover

1.3 Train/Validation

มีการทำ 80/20 แบบสุ่ม โดยนักศึกษาสุ่ม 1000 จุด ได้ Train size: 4055 และ Test size: 945

โดยมีข้อดีข้อเสียต่างกันดังนั้น
* แบบสุ่ม 80/20 ทำได้ง่าย ใช้ได้ดีกับข้อมูลทั่วไปและรันเร็ว แต่มีข้อเสียคือ จุดใกล้ๆกันมักมีลักษณะคล้ายกัน (spatial autocorrelation) ทำให้ตัว train กับ test เหมือนกัน
* Spatial Cross-validation มีข้อดีคือ ข้อมูลมีความเป็นจริงมากกว่า ลดปัญหาspatial autocorrelation แต่มีข้อเสียคือ ทำค่อนข้างยสากกว่าแบบสุ่ม ต้องมีการแบ่งพื้นที่ให้ดี ค่า Accuracy จะต่ำลง

1.4 Feature ที่ใช้

1. NDVI (Normalized Difference Vegetation Index) ใช้วัดพื้นที่ที่มีพืช
2. NDBI (Normalized Difference Built-up Index) ใช้ตรวจจับพื้นที่เมือง/สิ่งปลูกสร้าง
3. NDWI (Normalized Difference Water Index) ใช้ตรวจจับน้ำหรือความชื้นบนพื้นผิว
